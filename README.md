To-Do List App
A simple and clean To-Do List app built with HTML, CSS, and JavaScript.

Features
Add tasks
Mark tasks as complete ✅
Delete tasks ❌
Filter by All / Active / Completed
Task counter ("X items left")
localStorage persistence (tasks saved after page refresh)
How to Run
Clone this repository
   git clone https://github.com/DevipriyaRamesh97/todo-app.git
Open index.html in any browser
No dependencies or build steps required.

Project Structure
todo-app/

├── index.html → structure

├── style.css → styling

└── script.js → logic
